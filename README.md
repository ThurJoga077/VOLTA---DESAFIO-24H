# VOLTA — Embalagens Circulares para E-commerce

> Sistema de embalagens retornáveis inteligentes para reduzir o lixo e os custos do e-commerce brasileiro.

---

## Sobre o Projeto

O VOLTA é uma solução desenvolvida para o **Desafio 24 Horas de Inovação Global — INOVA CPS**, que propõe substituir embalagens descartáveis do e-commerce por um sistema circular de embalagens retornáveis com QR code.

O projeto foi desenvolvido **individualmente** por **Arthur Costa**, 16 anos, estudante da **Etec Lauro Gomes**.

---

## O Problema

- **87%** das embalagens de e-commerce vão direto para o lixo
- **R$ 2,8 bilhões** gastos por ano com embalagens descartáveis no Brasil
- **92%** do plástico de e-commerce não é efetivamente reciclado

O modelo atual é linear: usar e descartar. O VOLTA propõe mudar isso.

---

## A Solução

Embalagens resistentes com **QR code impresso**, que acompanham o pedido e retornam via Correios após a entrega. Cada embalagem dura **mais de 100 ciclos** após higienização industrial.

**Custo por uso:**
| Modelo | Custo |
|---|---|
| Embalagem descartável | R$ 8,50 |
| Embalagem VOLTA | R$ 1,20 |

Economia de **70 a 90%** por pedido.

---

## Como Funciona

1. Empresa assina o VOLTA e recebe as embalagens
2. Embala e envia o pedido normalmente
3. Cliente recebe e escaneia o QR code para devolver
4. Pack retorna via Correios
5. VOLTA higieniza e redistribui

---

## O Site

Landing page desenvolvida com **HTML, CSS e JavaScript puro**, sem frameworks.

**Funcionalidades:**
- Simulador interativo de economia por pedido
- Gráfico de impacto ambiental com **Chart.js**
- Captação de leads via **EmailJS**
- Animações com **Intersection Observer API**
- Contadores animados de impacto
- Menu mobile com hamburger animado
- Modo claro e escuro com `localStorage`
- Layout 100% responsivo — mobile, tablet e desktop

**Tecnologias:**
```
HTML5 · CSS3 · JavaScript ES6+
Chart.js · EmailJS · Google Fonts
```
